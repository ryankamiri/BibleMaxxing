# App Store Assets Guide

This folder stores App Store Connect handoff assets such as screenshot exports,
metadata checklists, and review-package supporting files.

Read the root `AGENTS.md` and `docs/AGENTS.md` before changing files here.

- Do not store Apple credentials, provisioning profiles, certificates, or demo
  account passwords here.
- Keep assets named by device class and screen purpose so App Store Connect
  uploads are easy to audit.
- Screenshots should show real BibleMaxxing UI, truthful source attribution, and
  no private user data.
